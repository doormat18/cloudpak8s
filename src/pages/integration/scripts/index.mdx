The capabilities provided with the Cloud Pak for Integration
can be installed using the instructions in 
the sections above in the 
Cloud Pak Playbook.

In addition, there is the option of installing and uninstalling
these capabilities using scripts developed by the IBM Garage
Solution Engineering team. Also available are Tekton pipelines
for installing and uninstalling the capabilities.

The scripts and pipeline artifacts, along with instructions
how to use them, are provided at the link below:

[Cloud Pak for Integration Installation Scripts & Pipelines](https://github.com/ibm-cloud-architecture/cp4i-setup/blob/master/README.md)
